# Template

## Dev

    npm install
    gulp

To run tests:

    gulp test

To watch for changes:

    gulp watch

To run ui:

    cd ui
    gup watch

To run clojure compiler:

    cd src/clojure
    lein cljsbuild auto

...or, if you have tmux installed, just run:

    watch.sh
